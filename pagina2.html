2.3. Relaciones de muchos a muchos.

las relaciones de muchos a muchos son un tipo de relacion de base de datos donde multiples registros de una tabla pueden estar asociados con multiples registros de otra tabla. un ejemplo mas practico seria el de una universidad ya que un estudiante puede inscribirse a varios cursos y un curso puede tener varios estudiantes y para manejar esto se crean 3 tablas una de estudiantes con su Id otra tabla con id del curso mas el nombre y otra con las inscripciones con la id de los estudiantes y el curso
